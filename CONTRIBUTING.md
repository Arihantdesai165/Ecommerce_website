# Contributing to E-commerce Website

First off, thank you for considering contributing to this project! It's people like you that make open source such a great community.

## Where do I go from here?

If you've noticed a bug or have a feature request, make sure to check our [Issues](../../issues) section to see if someone else in the community has already created a ticket. If not, go ahead and [make one](../../issues/new)!

## Fork & create a branch

If this is something you think you can fix, then [fork this repo](https://help.github.com/articles/fork-a-repo) and create a branch with a descriptive name.

## Get the test suite running

1. Install dependencies with `npm install`
2. Start the development server with `npm run dev`

## Commit your changes

Make your changes, test them, and commit them with a descriptive commit message. We recommend following the [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/) specification.

## Pull Request

Once you're finished with your changes, create a pull request, also known as a PR.
- Fill the PR template so that we can review your PR.
- Ensure the PR description clearly describes the problem and solution.
- Include the relevant issue number if applicable.

Thank you!
